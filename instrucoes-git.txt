FRENTE: git checkout -- nome-do-arquivo 
VERSO: Remove a alteração que foi feita no arquivo(que esteja trackeado) e que ainda não foi adicionada no stage.

FRENTE: git restore nome-do-arquivo
VERSO: Remove a alteração que foi feita no arquivo(que esteja trackeado) e que ainda não foi adicionada no stage.

FRENTE: git restore --staged nome-do-arquivo
VERSO: Remove o arquivo da stage area.

FRENTE: git fetch
VERSO: Verifica se as alterações realizadas no git local estão atualizadas com o git remoto e informa o usuário.

FRENTE: git tag nome-tag
VERSO: Usado para marcar versões dos projetos ou quando você deseja marca algo para encontrar mais fácilmente.

FRENTE: git push -tags
VERSO: Comando usado para enviar as tags para o git remote.

FRENTE: git branch nome-do-branch
VERSO: Comando usado para criar um novo branch

FRENTE: git branch -D nome-do-branch
VERSO: Comando que remove um branch local

FRENTE: git remote set-url origin url-desejada
VERSO: Comando altera a url/ssh cadastrado na variável origem, ex.: git remote set-url origin https://github.com/USERNAME/REPOSITORY.git
